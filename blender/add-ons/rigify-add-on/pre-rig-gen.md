# **Rigify (pre rig generate)**

- Enable it in blender add-on settings

# viewport

## Bone Axis

### Enable bone axis

- select metarig
- go to viewport display
- <img src="./images/pre-rig-gen/show-bone-rotation-axis.jpg" alt="show-bone-rotation-axis" />

### Bend Bone Direction

- in the pose mode
- select the bone
- set the Bend rotaion axis to x manual
- <img src="./images/pre-rig-gen/rigify-bend-rotation-axis.jpg" alt="rigify-bend-rotation-axis" />

### apply to all the selected bones

- select the bones
- change the `bone rotation axis` as per the above step
- right click as shown in below images (the red marked)
- <img src="./images/pre-rig-gen/bend-bone-rotation-copy-selected.jpg" />
- select `Copy to selected`
- repeat the same on the other side (like the left or the right side)
