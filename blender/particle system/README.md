## Particle edit mode
- select the number of segments and number of hairs
- seed - random distribution and positioning of the strands

### move, rotate, scale
- <kbd>G</kbd> - Move
- <kbd>R</kbd> - Rotate
- <kbd>S</kbd> - Scale
